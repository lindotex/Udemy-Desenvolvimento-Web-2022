Estilo CSS (Cascading Style Sheets)

Classe

Pode ser Utilizado para classificar um grupo de valores ou dados semelhantes

Ex: Passageiros Homens, em um onibus, podem ser "classificados" por "Masculino", e as passageiras por "Feminino". Assim todos os passageiros ocupam apenas duas classes, mas já facilita na organização dos dados.

ID

Deve ser Utilizado para classificar uma unidade de valor ou dado

Ex: Passageiros, em um onibus, podem ser "classificados" por "Masculino" e "Feminino". Mas se quizermos localizar um individuo específico, o ideal seria termos um numero "Identificador", assim como o acento... que é individual e único.

ID > Classe:

No casso de sobreposição de Class e ID, o componente será estilizado de acordo com a ID, ja que o ID é um componente individual dentro da Classe.
